# Prácticas de Java

Este repositorio contiene ejercicios básicos desarrollados en **Java** con fines académicos y didácticos.

El objetivo principal es practicar la programación en Java desde cero, incorporando conceptos fundamentales del lenguaje y aprendiendo a trabajar con estructuras de datos simples.

## ¿Qué estoy practicando?

En estos programas estoy aprendiendo y reforzando conceptos como:

- Sintaxis básica de Java
- Creación de clases
- Uso del método `main`
- Declaración de variables
- Uso de listas con `ArrayList`
- Agregado de elementos a una lista
- Acceso al primer y último elemento
- Recorrido de listas
- Uso de `stream()`
- Filtrado de datos con `filter()`
- Ordenamiento de elementos con `sorted()`
- Conteo de elementos con `count()`
- Uso de condicionales `if` y `else`
- Impresión de resultados por consola

## Descripción

Los ejercicios de este repositorio están orientados a comprender cómo se trabaja con listas en Java y cómo se pueden aplicar operaciones sobre sus elementos.

Actualmente se practican operaciones como:

- Crear una lista de nombres
- Agregar elementos a la lista
- Mostrar elementos por consola
- Filtrar nombres según una condición
- Ordenar nombres alfabéticamente
- Contar elementos que cumplen una condición
- Obtener el primer y último elemento de una lista
